# Mastermind
Juego de Mastermind
Código secreto de 6 dígitos (0–9), sin dígitos repetidos.
   - Dos jugadores, turnos alternos (pass-and-play).
   - 10 intentos por jugador.
   - Feedback por intento: exactos (posición y dígito correctos) y parciales (dígito está pero en otra posición). mostrar exactos de forma visual con una bola verde, y parciales con una bola roja
   - Gana quien acierte con menos intentos; si nadie acierta, gana quien tenga más exactos (si empatan, más parciales)
